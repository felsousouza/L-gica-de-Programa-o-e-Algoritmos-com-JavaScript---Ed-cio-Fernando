# Projeto de Estudo

Este repositório contém exercícios de aprendizado e prática em desenvolvimento web.

## Observação importante

A partir do último item de exercício realizado hoje, **todas as estilizações (CSS) e o HTML** dos próximos exercícios serão gerados por Inteligência Artificial. O objetivo é simular cenários mais reais e próximos do que se encontra em empresas reais, onde a IA auxilia na criação de interfaces e na padronização de código.
